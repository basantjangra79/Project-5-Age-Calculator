# 🎂 Age Calculator

A simple Python Age Calculator that calculates a user's current age based on their Date of Birth (DOB).

This project is built using **Python Functions**, **Datetime Module**, **Loops**, and **Exception Handling** to create a clean and user-friendly command-line application.

---

## 🚀 Features

* 📅 Calculate age from Date of Birth
* ✅ DOB validation using `datetime.strptime()`
* 🔄 Continuous execution with loop support
* ⚠️ Handles invalid date formats
* 🛡️ Exception handling for user errors
* ⌨️ Supports graceful exit using `Ctrl + C`
* 🧩 Function-based program structure

## 🔍 How Age is Calculated

The program:

1. Takes the user's DOB.
2. Extracts the birth year, month, and day.
3. Compares it with the current date.
4. Checks whether the birthday has occurred this year.
5. Calculates the correct age accordingly.
